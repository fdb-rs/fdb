Module fdb::database[−][src]

Expand description

Provides FdbDatabase type for working with FDB Database.

Clients operating on a FdbDatabase should, in most cases use the run method. This implements a proper retry loop around the work that needs to be done and, assure that commit has returned successfully before returning.
